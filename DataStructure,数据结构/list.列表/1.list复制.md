
# list 复制

复制列表（或多数内置的可变集合）最简单的方式是使用内置的类型构造方法。例如：
```py
>>> l1 = [3, [55, 44], (7, 8, 9)]
>>> l2 = list(l1)  ➊
>>> l2
[3, [55, 44], (7, 8, 9)]
>>> l2 == l1  ➋
True
>>> l2 is l1  ➌
False
```
❶ list(l1) 创建 l1 的副本。

❷ 副本与源列表相等。

❸ 但是二者指代不同的对象。对列表和其他可变序列来说，还能使用简洁的 l2 = l1[:] 语句创建副本。

然而，构造方法或 [:] 做的是浅复制（即复制了最外层容器，副本中的元素是源容器中元素的引用）。
如果所有元素都是不可变的，那么这样没有问题，还能节省内存。
但是，如果有可变的元素，可能就会导致意想不到的问题。
