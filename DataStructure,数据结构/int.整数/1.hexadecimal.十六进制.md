## 十六进制切割

### method.1
```py
>>> i = int("0x221f2104", 16)
>>> h = hex(i)
>>> b = bin(i)
>>> b[-8:]
'00000100'
>>> b[-16:-8]
'00100001'
>>> b[-24:-16]
'00011111'
>>> b[:-24]
'0b100010'
```

```py
>>> hex(int(b[:-24], 2))
'0x22'
>>> hex(int(b[-24:-16], 2))
'0x1f'
>>> hex(int(b[-16:-8], 2))
'0x21'
>>> hex(int(b[-8:], 2))
'0x4'
```


### method.2
```py
>>> i = int("0x221f2104", 16)
>>> h = hex(i)
>>> a1 = divmod(i, 0x100)
>>> a2 = divmod(2236193, 0x100)
>>> a3 = divmod(8735, 0x100)
>>> a4 = divmod(34, 0x100)
>>> a1
(2236193, 4)
>>> a2
(8735, 33)
>>> a3
(34, 31)
>>> a4
(0, 34)
>>> hex(4)
'0x4'
>>> hex(33)
'0x21'
>>> hex(31)
'0x1f'
>>> hex(34)
'0x22'
```


### method.3

```py
>>> i
572465412
>>> hex(i)
'0x221f2104'
>>> i_0 = i
>>> i_0
572465412
>>> i_1 = i_0 >> 8
>>> i_2 = i_1 >> 8
>>> i_3 = i_2 >> 8
>>> i_0 & int(0b11111111)
4
>>> i_1 & int(0b11111111)
33
>>> i_2 & int(0b11111111)
31
>>> i_3 & int(0b11111111)
34
>>> hex(i_3 & int(0b11111111))
'0x22'
>>> hex(i_2 & int(0b11111111))
'0x1f'
>>> hex(i_1 & int(0b11111111))
'0x21'
>>> hex(i_0 & int(0b11111111))
'0x4'
```
