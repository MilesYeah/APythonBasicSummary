
# lambda 匿名函数

```py
>>> func = lambda x:x**2
>>> func(2)
4
>>> func(3)
9
>>>
>>> func = lambda x,y:x*y
>>> func(1,2)
2
>>> func(2,2)
4
>>> func(3,2)
6
>>> func(3)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: <lambda>() missing 1 required positional argument: 'y'
>>>
```
