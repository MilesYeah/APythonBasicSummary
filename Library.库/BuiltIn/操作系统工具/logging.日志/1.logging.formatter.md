
# logging.formatter





### format常用格式说明

| 参数            | 作用                                           |
| --------------- | ---------------------------------------------- |
| %(asctime)s     | 年-月-日 时-分-秒,毫秒 2013-04-26 20:10:43,745 |
| %(filename)s    | 文件名，不含目录                               |
| %(pathname)s    | 目录名，完整路径                               |
| %(funcName)s    | 函数名                                         |
| %(levelno)s     | 打印日志级别的数值                             |
| %(levelname)s   | 级别名                                         |
| %(lineno)d      | 行号                                           |
| %(module)s      | 模块名                                         |
| %(message)s     | 消息体                                         |
| %(name)s        | 日志模块名                                     |
| %(process)d     | 进程id                                         |
| %(processName)s | 进程名                                         |
| %(thread)d      | 线程id                                         |
| %(threadName)s  | 线程名                                         |
