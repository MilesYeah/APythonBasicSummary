# 身份运算符.IdentityOperator

身份运算符用于比较两个对象的存储单元
| 运算符   | 描述                                 |
| -------- | ------------------------------------ |
| `is`     | 判断两个标识符是不是引用自`相同`对象 |
| `is not` | 判断两个标识符是不是引用自`不同`对象 |


1. `x is y`
   1. 类似 id(x) == id(y) , 
   2. 如果引用的是同一个对象则返回 True，
   3. 否则返回 False
2. `x is not y`
   1. 类似 id(a) != id(b)。
   2. 如果引用的不是同一个对象则返回结果 True，
   3. 否则返回 False

注： id() 函数用于获取对象内存地址。
