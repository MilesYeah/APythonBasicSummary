# 成员运算符
除了以上的一些运算符之外，Python还支持成员运算符，测试实例中包含了一系列的成员，包括字符串，列表或元组。

| 运算符   | 描述                                                    |
| -------- | ------------------------------------------------------- |
| `in`     | 在指定序列中**找到**值返回 True，<br>否则返回 False。   |
| `not in` | 在指定序列中**未找到**值返回 True，<br>否则返回 False。 |
